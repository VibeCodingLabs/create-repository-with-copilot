# Architecture

## System Overview

This repository follows the **ai-platform-scaffold-forge** architecture:

- **Frontend**: Next.js 16 App Router + Vite Dashboards
- **Backend**: Hono + OpenAPI 3.1 + Zod
- **AI Layer**: Vercel AI SDK + AI Gateway
- **Agent System**: Multi-agent with Planner-Executor-Verifier pattern
- **Workflows**: 8 reusable blueprints (YAML + JSON)
- **Database**: SQLite with CLI tools

## Key Principles

1. **Copilot-First Development**
2. **Type Safety** (Strict TypeScript + Zod)
3. **Real Data** (No mocks in production paths)
4. **Composable Agents**
5. **Observable Workflows**

## Folder Structure

- `src/app` → Next.js pages & layouts
- `src/api` → Hono routes
- `src/agents` → Agent implementations
- `src/workflows` → Workflow blueprints
- `evals` → Evaluation tests
